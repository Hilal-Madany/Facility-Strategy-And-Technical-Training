# 📑 Technical Quotation & BOQ Protocol
*Derived from Accumen Techno Marketing Solution Framework*

### Standard 10-Step Quotation Structure
1. **Ref & Date** | 2. **Subject** | 3. **Introduction** | 4. **Technical Scope** | 5. **Commercial BOQ**
6. **Taxes (GST)** | 7. **Exclusions** | 8. **Payment Terms** | 9. **Warranty** | 10. **Validity**

### Sample Industrial BOQ (MS Ducting Project)
| Item Description | Qty | Unit | Rate (INR) | Total (INR) |
| :--- | :--- | :--- | :--- | :--- |
| MS Duct 3mm dia 520mm | 10 | Mtr | 4,148 | 41,475 |
| MS Hopper 800x1000 H-900 | 7 | Nos | 13,125 | 91,875 |
| Erection & Commissioning | 1 | Lot | 65,750 | 65,750 |
| **Grand Total (Inc. GST)** | | | | **3,59,846** |

> **Key Rule:** The *Exclusions* clause is as critical as the scope. Always define what is NOT included (e.g., civil work, scaffolding).
